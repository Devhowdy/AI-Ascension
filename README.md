# AI Battle Arena

Cinematic frontend-only React/Vite prototype for an original AI battle game.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router
- Local state only

## Pages

- Home
- Character Select
- Battle Arena

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
